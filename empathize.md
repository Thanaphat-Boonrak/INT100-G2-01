## empathize

### 1st interview information

ลูกชายของคุณลุงเรียนอยู่ที่ไหนครับ?
```
ลูกชายของผมศึกษาที่มหาวิทยาลัยพระจอมเกล้าธนบุรีครับ เขาเรียนในสาขาวิศวกรรมศาสตร์และเพิ่งจบการศึกษาไปไม่นานมานี้ครับ
```
คุณลุงมาที่นี่บ่อยไหมครับ?
```
ไม่ได้มาทุกวันครับ แต่ก็ถือว่ามาเป็นประจำครับ
```
คุณลุงมาที่นี่เพื่อทำกิจกรรมอะไรครับ?
```
ผมมาที่นี่เพื่อเดินออกกำลังกายครับ
```
บ้านของคุณลุงอยู่แถวนี้ใช่ไหมครับ?
```
ใช่ครับ บ้านของผมห่างจากศูนย์กีฬาไปประมาณสองกิโลเมตรครับ
```
คุณลุงเดินทางมาที่นี่อย่างไรครับ?
```
ผมขับรถมาครับ
```
คุณลุงออกกำลังกายเพื่อเหตุผลอะไรครับ? เพื่อสุขภาพหรือเพื่อสร้างกล้ามเนื้อ?
```
ผมออกกำลังกายเพื่อสุขภาพครับ
```
ตอนนี้คุณลุงอายุประมาณเท่าไหร่ครับ?
```
ผมอายุประมาณ 70 ปีครับ
```
คุณลุงมาใช้บริการที่นี่มานานแล้วหรือยังครับ?
```
ผมมาที่นี่มาประมาณ 10 ปีแล้วครับ
```
เมื่อ 10 ปีที่แล้ว ที่นี่มีการเปลี่ยนแปลงอะไรบ้างไหมครับ?
```
ตอนนี้มีการติดตั้งแสงไฟสปอร์ตไลท์เพิ่มเติม และความสะอาดก็ดีขึ้นมากครับ
```
คุณลุงพบเจอปัญหาอะไรบ้างไหมครับ?
```
ปัญหาที่พบคือรถที่เข้ามาในบริเวณนี้มักขับด้วยความเร็วสูง ควรมีการชะลอความเร็ว และไฟสูงของรถทำให้มองไม่ชัดสำหรับผู้สูงอายุครับ
```
คุณลุงมีข้อเสนอแนะอะไรเพิ่มเติมไหมครับ?
```
อยากให้มีการบริหารจัดการศูนย์กีฬาให้ดีขึ้นครับ
```
งานวิจัยนี้มีวัตถุประสงค์เพื่ออะไรครับ?
```
งานวิจัยนี้มุ่งเน้นการออกแบบ โดยเราต้องสัมภาษณ์ผู้ใช้เพื่อค้นหาจุดที่ต้องการการปรับปรุง และนำข้อมูลเหล่านั้นไปพัฒนาในขั้นตอนถัดไปครับ
```
คุณลุงรู้จักชมรมหรือชุมชนที่เกี่ยวข้องกับกิจกรรมที่คุณลุงทำไหมครับ?
```
รู้จักครับ มีชมรมหลายแห่งและชุมชนของผู้สูงอายุที่เข้าร่วมกิจกรรมกัน
```
คุณลุงมีความคิดเห็นเพิ่มเติมอะไรไหมครับ?
```
อยากให้มีการปรับปรุงและบริหารจัดการที่ดีขึ้นครับ
```
ขอบคุณมากครับ
```
ขอบคุณครับ
```

### what

- รถยนต์เข้ามาในพื้นที่สำหรับวิ่งและขับด้วยความเร็ว ไฟสูงจากรถที่อาจทำให้มองไม่ชัด
- ต้องการการบริหารจัดการที่ดีขึ้นสำหรับศูนย์กีฬา

### why

- รถยนต์ที่เข้ามาในบริเวณสำหรับวิ่งและขับด้วยความเร็วสูง ทำให้เกิดปัญหาการมองเห็นไม่ชัดเจนโดยเฉพาะสำหรับคนสูงอายุที่เจอไฟสูงจากรถ

### where

- ศูนย์กีฬา

### how

- ควรมีการจัดการที่ดีขึ้น เช่น การลดความเร็วรถยนต์ในพื้นที่สำหรับวิ่ง และการจัดการศูนย์กีฬาให้ดียิ่งขึ้น

### 2st interview information

พี่มาที่นี่บ่อยไหมครับ?
```
ปกติแล้วจะมาที่นี่ประมาณสัปดาห์ละสองถึงสามครั้งครับ
```
แล้วพี่มาที่นี่เพื่อทำกิจกรรมอะไรครับ?
```
ส่วนใหญ่แล้วผมมาที่นี่เพื่อว่ายน้ำครับ
```
พี่เริ่มว่ายน้ำมานานแล้วหรือยังครับ?
```
ผมเริ่มว่ายน้ำมาประมาณ 3 ปีแล้วครับ
```
อะไรเป็นเหตุผลที่ทำให้พี่เริ่มว่ายน้ำครับ? พี่มีแรงบันดาลใจหรือเหตุผลพิเศษอะไรไหม?
```
ผมเริ่มว่ายน้ำเพราะมีความกลัวว่าจะตกน้ำครับ ซึ่งทำให้ผมรู้สึกว่าต้องเรียนรู้วิธีการว่ายน้ำ
```
เมื่อพี่มาที่นี่ พี่มุ่งหวังเรื่องสุขภาพหรือการออกกำลังกายด้วยไหมครับ?
```
ผมมาที่นี่เพื่อว่ายน้ำทั้งเพื่อสุขภาพและเพื่อการออกกำลังกายครับ
```
พี่มีเป้าหมายเฉพาะในการออกกำลังกายหรือไม่ครับ?
```
เป้าหมายหลักของผมคือการรักษาสุขภาพและความแข็งแรงครับ
```
พี่มาที่นี่ช่วงเวลาไหนและกลับประมาณกี่โมงครับ?
```
เวลาไม่ค่อยแน่นอนครับ บางวันผมมาช่วงเช้าหรือช่วงบ่าย แต่วันนี้มาที่นี่ช่วงเย็นครับ
```
เมื่อ 3 ปีที่แล้วที่พี่มาที่นี่ มีการเปลี่ยนแปลงอะไรบ้างไหมครับ?
```
เมื่อ 3 ปีที่แล้วมีการปรับปรุงเรื่องการติดตั้งไฟและการทำสีพื้นใหม่ครับ ตอนนั้นยังไม่มีการเปลี่ยนแปลงอื่น ๆ มากมาย
```
ถ้าพี่มีความคิดเห็นเกี่ยวกับการปรับปรุงที่นี่ พี่คิดว่าควรมีการปรับปรุงอะไรเพิ่มเติมไหมครับ?
```
ผมคิดว่าควรมีการจัดการเรื่องรถที่เข้ามาในบริเวณที่ใช้สำหรับการวิ่งครับ เพราะไม่ควรมีรถเข้าไปในพื้นที่นั้น
```
ขอบคุณพี่มากครับ
```
ขอบคุณครับ
```

### what

- มีการเข้าออกของรถยนต์เข้าไปในพื้นที่สำหรับคนออกกำลังกาย
- พื้นถนนที่ขรุขระหรือทางเดินที่เป็นหลุมเป็นล่องในการออกกำลังกาย

### why

- มีการเข้าออกของรถยนต์ในที่สำหรับคนวิ่งและเดินควรเป็นที่ปลอดภัยสำหรับผู้ใช้บริการที่ต้องการออกกำลังกาย
- เกิดจากไม่มีการดูแลหรือสนใจ

### where

- ศูนย์กีฬา

### how

- ติดตั้งป้ายเตือนสำหรับบริเวณที่อาจเกิดอุบัติเหตุได้ เช่น เขตห้ามเข้า
- ตรวจสอบและบำรุงรักษาพื้นที่อย่างสม่ำเสมอ เช่น การซ่อมแซมพื้นผิว

### 3rd interview information

สวัสดีครับพี่ มาที่นี่บ่อยไหมครับ?
```
สวัสดีครับ ผมมาที่นี่เกือบทุกวันครับ
```
พี่มาที่นี่มานานแค่ไหนแล้วครับ?
```
มาที่นี่มาประมาณสองปีเศษแล้วครับ
```
เมื่อสองปีที่แล้ว ที่นี่มีการเปลี่ยนแปลงอะไรบ้างไหมครับ?
```
มีการพัฒนาหลายอย่างครับ เช่น สภาพแวดล้อมที่ดีขึ้น การเพิ่มความสนใจในการออกกำลังกายของผู้คน และการปรับปรุงเครื่องอุปกรณ์ให้ดีขึ้น
```
พี่มีข้อเสนอแนะอะไรที่อยากให้ปรับปรุงในยิมนี้ไหมครับ?
```
อยากให้เครื่องออกกำลังกายมีความทันสมัยมากขึ้นครับ
```
ในช่วงเวลาที่ผ่านมา พี่พบเจอปัญหาอะไรบ้างไหมครับ?
```
บางครั้งอาจมีปัญหาที่เกี่ยวกับธรรมชาติ เช่น สภาพอากาศ หรือปัญหากับอุปกรณ์ที่พังครับ นอกจากนี้ยังมีสิ่งที่อาจรบกวนการใช้งานบ้าง
```
เข้าใจครับ พี่พูดถึงการปรับปรุงต่าง ๆ เพราะมีการศึกษาเกี่ยวกับการออกแบบใช่ไหมครับ?
```
ใช่ครับ การพูดถึงการปรับปรุงเป็นส่วนหนึ่งของการคิดเพื่อการออกแบบ ผมต้องการเห็นเครื่องออกกำลังกายที่ทันสมัยยิ่งขึ้น รวมถึงการปรับปรุงสภาพแวดล้อมให้ดียิ่งขึ้นครับ
```
การออกกำลังกายของพี่มีเป้าหมายอะไรบ้างครับ?
```
เป้าหมายหลักของผมคือการสร้างความแข็งแรงให้กับตัวเอง ดูดี และเพิ่มความมั่นใจ รวมถึงสร้างแรงบันดาลใจให้กับผู้อื่นครับ
```
ขอทราบอายุของพี่ได้ไหมครับ?
```
ตอนนี้ผมอายุ 27 ปีครับ
```
ขอบคุณมากครับพี่
```
ขอบคุณครับ สวัสดีครับ
```


### what

- ความไม่ทันสมัยของเครื่องเล่น
- หรือเครื่องออกกำลังกาย
- การที่ไม่มีคนมายอมดูแล
- ความไม่สะอาดของสวนธน

### why

- เนื่องจากตัวเราเองไม่มีอำนาจและงบประมาณที่จะทำอะไรด้วยตัวเองโดยพลการ เพราะเป็นของสาธาณะ
- คนไม่ยอมทิ้งขยะให้เป็นที่

### where

- สวนธนบุรีรมย์

### how

- ติดตั้งถังขยะเพิ่มเติมในจุดที่เหมาะสม และจัดให้มีป้ายเตือน
- จัดทำงบประมาณหรือขอระดุมทุนเพื่ออุปกรณ์ที่ดี
