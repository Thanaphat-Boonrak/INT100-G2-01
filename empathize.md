## empathize

### Interview information (1)

สวัสดีครับพี่ มาที่นี่บ่อยไหมครับ?
```
สวัสดีครับ ผมมาที่นี่เกือบทุกวันครับ
```
พี่มาที่นี่มานานแค่ไหนแล้วครับ?
```
มาที่นี่มาประมาณสองปีเศษแล้วครับ
```
เมื่อสองปีที่แล้ว ที่นี่มีการเปลี่ยนแปลงอะไรบ้างไหมครับ?
```
มีการพัฒนาหลายอย่างครับ เช่น สภาพแวดล้อมที่ดีขึ้น การเพิ่มความสนใจในการออกกำลังกายของผู้คน และการปรับปรุงเครื่องอุปกรณ์ให้ดีขึ้น
```
พี่มีข้อเสนอแนะอะไรที่อยากให้ปรับปรุงในยิมนี้ไหมครับ?
```
อยากให้เครื่องออกกำลังกายมีความทันสมัยมากขึ้นครับ
```
ในช่วงเวลาที่ผ่านมา พี่พบเจอปัญหาอะไรบ้างไหมครับ?
```
บางครั้งอาจมีปัญหาที่เกี่ยวกับธรรมชาติ เช่น สภาพอากาศ หรือปัญหากับอุปกรณ์ที่พังครับ นอกจากนี้ยังมีสิ่งที่อาจรบกวนการใช้งานบ้าง
```
เข้าใจครับ พี่พูดถึงการปรับปรุงต่าง ๆ เพราะมีการศึกษาเกี่ยวกับการออกแบบใช่ไหมครับ?
```
ใช่ครับ การพูดถึงการปรับปรุงเป็นส่วนหนึ่งของการคิดเพื่อการออกแบบ ผมต้องการเห็นเครื่องออกกำลังกายที่ทันสมัยยิ่งขึ้น รวมถึงการปรับปรุงสภาพแวดล้อมให้ดียิ่งขึ้นครับ
```
การออกกำลังกายของพี่มีเป้าหมายอะไรบ้างครับ?
```
เป้าหมายหลักของผมคือการสร้างความแข็งแรงให้กับตัวเอง ดูดี และเพิ่มความมั่นใจ รวมถึงสร้างแรงบันดาลใจให้กับผู้อื่นครับ
```
ขอทราบอายุของพี่ได้ไหมครับ?
```
ตอนนี้ผมอายุ 27 ปีครับ
```
ขอบคุณมากครับพี่
```
ขอบคุณครับ สวัสดีครับ
```

**What:** ความไม่ทันสมัยของเครื่องเล่นหรือเครื่องออกกำลังกาย, การที่ไม่มีคนมายอมดูแล, ความไม่สะอาดของสวนธน

**Why:** เนื่องจากตัวเราเองไม่มีอำนาจและงบประมาณที่จะทำอะไรด้วยตัวเองโดยพลการ, เพราะเป็นของสาธาณะ, คนไม่ยอมทิ้งขยะให้เป็นที่

**Where:** สวนธนบุรีรมย์

**How:** ติดตั้งถังขยะเพิ่มเติมในจุดที่เหมาะสม และจัดให้มีป้ายเตือน, จัดทำงบประมาณหรือขอระดุมทุนเพื่ออุปกรณ์ที่ดี

#### _Say_

  - "มาเกือบทุกวันกับทุกวัน และฉันมาที่นี่นาน"
  - "ประมาณ 2 ปี"
  - "มีสภาพแวดล้อมที่ดีขึ้นแล้วก็มีคนที่สนใจเรื่องของการออกกำลังกาย"
  - "เครื่องอุปกรณ์เนี่ยก็ปรับปรุงในทางที่ดีขึ้น"
  - "อยากให้เครื่องเล่นมันทันสมัยมากขึ้น"
  - "อยากให้มีสภาพแวดล้อมปลอดโปร่งกว่านี้"
  - "อยากให้มีธรรมชาติมากกว่านี้"
  - "เป้าหมายของการมาออกกำลังกายคือความแข็งแรงและการดูดี"

#### _Do_

  - มาที่ศูนย์กีฬาเกือบทุกวันและอยู่ที่นี่ประมาณ 2 ปี
  - เข้ามาใช้บริการในช่วงเวลาไม่แน่นอน เช่น เช้า บ่าย หรือเย็น
  - ใช้อุปกรณ์ที่ศูนย์กีฬาและวางแผนที่จะออกกำลังกายเพื่อความแข็งแรงและการเสริมบุคลิกภาพ
  - สังเกตเห็นการปรับปรุงของศูนย์กีฬา เช่น สภาพแวดล้อมและอุปกรณ์
  - แสดงความต้องการให้มีการปรับปรุงอุปกรณ์ให้ทันสมัยและสภาพแวดล้อมให้ดีขึ้น

#### _Think_

  - คิดว่าศูนย์กีฬามีการปรับปรุงในบางด้าน เช่น สภาพแวดล้อมและอุปกรณ์ แต่ยังมีบางส่วนที่ต้องการการพัฒนาเพิ่มเติม
  - คิดว่าการออกกำลังกายมีเป้าหมายที่สำคัญในการเสริมสร้างความแข็งแรงและการเสริมบุคลิกภาพ
  - คิดว่าการปรับปรุงเครื่องเล่นให้ทันสมัยและการเพิ่มธรรมชาติในสภาพแวดล้อมจะช่วยให้มีประสบการณ์การออกกำลังกายที่ดีขึ้น
  - คิดว่าความสะอาดและความปลอดภัยในศูนย์กีฬาควรได้รับการจัดการอย่างดี

#### _Feel_

  - รู้สึกพอใจที่ศูนย์กีฬาได้มีการปรับปรุงบางอย่างแล้ว แต่ยังรู้สึกว่ามีพื้นที่ที่ควรได้รับการพัฒนาเพิ่มเติม
  - รู้สึกว่าการออกกำลังกายที่ศูนย์กีฬาเป็นสิ่งที่สำคัญสำหรับการเสริมความแข็งแรงและบุคลิกภาพ
  - รู้สึกว่าควรมีสภาพแวดล้อมที่ดีและปลอดโปร่งมากขึ้น รวมถึงการมีธรรมชาติเพิ่มเติมเพื่อให้รู้สึกผ่อนคลายมากขึ้น
  - รู้สึกไม่สะดวกและเป็นกังวลเกี่ยวกับการเข้ามาของรถยนต์ในพื้นที่สำหรับวิ่ง ซึ่งอาจทำให้เกิดความไม่ปลอดภัย

### Interview information (2)

ลูกชายของคุณลุงเรียนอยู่ที่ไหนครับ?

```
ลูกชายของผมศึกษาที่มหาวิทยาลัยพระจอมเกล้าธนบุรีครับ เขาเรียนในสาขาวิศวกรรมศาสตร์และเพิ่งจบการศึกษาไปไม่นานมานี้ครับ
```
คุณลุงมาที่นี่บ่อยไหมครับ?

บ้านของคุณลุงอยู่แถวนี้ใช่ไหมครับ?
```
ใช่ครับ บ้านของผมห่างจากศูนย์กีฬาไปประมาณสองกิโลเมตรครับ
```

คุณลุงเดินทางมาที่นี่อย่างไรครับ?

```
ผมขับรถมาครับ
```

```
ผมออกกำลังกายเพื่อสุขภาพครับ
```

```
ผมอายุประมาณ 70 ปีครับ
```

คุณลุงมาใช้บริการที่นี่มานานแล้วหรือยังครับ?

```
ผมมาที่นี่มาประมาณ 10 ปีแล้วครับ
```

เมื่อ 10 ปีที่แล้ว ที่นี่มีการเปลี่ยนแปลงอะไรบ้างไหมครับ?

```
ตอนนี้มีการติดตั้งแสงไฟสปอร์ตไลท์เพิ่มเติม และความสะอาดก็ดีขึ้นมากครับ
```
คุณลุงพบเจอปัญหาอะไรบ้างไหมครับ?
```
ปัญหาที่พบคือรถที่เข้ามาในบริเวณนี้มักขับด้วยความเร็วสูง ควรมีการชะลอความเร็ว และไฟสูงของรถทำให้มองไม่ชัดสำหรับผู้สูงอายุครับ
```

```
รู้จักครับ มีชมรมหลายแห่งและชุมชนของผู้สูงอายุที่เข้าร่วมกิจกรรมกัน
```

ขอบคุณมากครับ
```
ขอบคุณครับ
```

**What:** รถยนต์เข้ามาในพื้นที่สำหรับวิ่งและขับด้วยความเร็ว ไฟสูงจากรถที่อาจทำให้มองไม่ชัด, ต้องการการบริหารจัดการที่ดีขึ้นสำหรับศูนย์กีฬา

**Why:** รถยนต์ที่เข้ามาในบริเวณสำหรับวิ่งและขับด้วยความเร็วสูง ทำให้เกิดปัญหาการมองเห็นไม่ชัดเจนโดยเฉพาะสำหรับคนสูงอายุที่เจอไฟสูงจากรถ

**Where:** ศูนย์กีฬา

**How:** ควรมีการจัดการที่ดีขึ้น เช่น การลดความเร็วรถยนต์ในพื้นที่สำหรับวิ่ง และการจัดการศูนย์กีฬาให้ดียิ่งขึ้น

#### _Say_

  - "ไม่ทุกวัน แต่ประจำ"
  - "ก็คือมาเดินออกกำลังกาย"
  - "เป้าหมายในการออกกำลังกายก็เพื่อสุขภาพแหละ"
  - "ตอนนี้ก็อายุประมาณ 70 ปีแล้ว"
  - "มาที่นี่ประมาณ 10 ปีแล้ว"
  - "ปัญหาที่เจอก็คงจะเป็นรถที่เข้ามาในบริเวณสำหรับวิ่ง"
  - "ขับมาด้วยความเร็ว ควรจะชะลอกว่านี้"

#### _Do_

  - มาที่ศูนย์กีฬาเพื่อเดินออกกำลังกายเป็นประจำ
  - ขับรถมาจากบ้านซึ่งห่างจากศูนย์กีฬาประมาณ 1 กิโลเมตร
  - ใช้เวลาที่ศูนย์กีฬาเพื่อรักษาสุขภาพ
  - สังเกตเห็นการเปลี่ยนแปลงในศูนย์กีฬา เช่น การติดตั้งแสงไฟใหม่

#### _Think_

  - การรักษาสุขภาพและการออกกำลังกายมีความสำคัญมาก แม้ว่าจะอายุมากแล้ว
  - คิดว่าการขับรถในพื้นที่ที่มีคนวิ่งควรชะลอความเร็วเพื่อความปลอดภัย
  - การมีแสงไฟที่ดีมีความสำคัญต่อความปลอดภัยของคนที่มาใช้บริการ โดยเฉพาะผู้สูงอายุ
  - คิดว่าการบริหารจัดการศูนย์กีฬาอาจยังมีข้อบกพร่องที่ต้องปรับปรุง

#### _Feel_

  - รู้สึกว่าการออกกำลังกายเป็นสิ่งที่สำคัญและมีผลดีต่อสุขภาพ
  - รู้สึกกังวลเกี่ยวกับความปลอดภัยจากรถยนต์ที่เข้ามาในพื้นที่สำหรับวิ่ง
  - รู้สึกว่าแสงไฟไม่เพียงพอและอาจทำให้มองไม่ชัด
  - รู้สึกว่าอยากให้ศูนย์กีฬามีการบริหารจัดการที่ดีขึ้นเพื่อความสะดวกและปลอดภัย


```
ผมคิดว่าควรมีการจัดการเรื่องรถที่เข้ามาในบริเวณที่ใช้สำหรับการวิ่งครับ เพราะไม่ควรมีรถเข้าไปในพื้นที่นั้น
```
ขอบคุณพี่มากครับ
```
ขอบคุณครับ
```

**What:** มีการเข้าออกของรถยนต์เข้าไปในพื้นที่สำหรับคนออกกำลังกาย, พื้นถนนที่ขรุขระหรือทางเดินที่เป็นหลุมเป็นล่องในการออกกำลังกาย

**Why:** มีการเข้าออกของรถยนต์ในที่สำหรับคนวิ่งและเดิน ควรเป็นที่ปลอดภัยสำหรับผู้ใช้บริการที่ต้องการออกกำลังกาย, เกิดจากไม่มีการดูแลหรือสนใจ

**Where:** ศูนย์กีฬา

**How:** ติดตั้งป้ายเตือนสำหรับบริเวณที่อาจเกิดอุบัติเหตุได้ เช่น เขตห้ามเข้า, ตรวจสอบและบำรุงรักษาพื้นที่อย่างสม่ำเสมอ เช่น การซ่อมแซมพื้นผิว

#### _Say_

  - "อาทิตย์ละ 2-3 ครั้ง"
  - "ส่วนใหญ่จะมาว่ายน้ำ"
  - "มาที่นี่ประมาณ 3 ปี"
  - "กลัวตัวเองตกน้ำแล้วช่วยเหลือตัวเองไม่ได้"
  - "เป้าหมายของการมาว่ายน้ำคือเอาชีวิตรอดได้เมื่อจมน้ำ และความแข็งแรงสุขภาพดี"

#### _Do_

  - มาที่ศูนย์กีฬาเป็นประจำประมาณอาทิตย์ละ 2-3 ครั้ง
  - ว่ายน้ำเพื่อรักษาสุขภาพและเรียนรู้ทักษะการว่ายน้ำ
  - ใช้เวลาว่ายน้ำเพื่อเตรียมความพร้อมในการช่วยเหลือตัวเองในกรณีที่ตกน้ำ
  - มีการมาศูนย์กีฬาในช่วงเวลาที่ไม่แน่นอน เช่น เช้า บ่าย หรือเย็น
  - สังเกตเห็นการเปลี่ยนแปลงในศูนย์กีฬา เช่น การเปลี่ยนแสงไฟและทำสีพื้นสำหรับวิ่งใหม่

#### _Think_

  - การว่ายน้ำเป็นวิธีที่สำคัญในการเตรียมตัวสำหรับสถานการณ์ฉุกเฉิน เช่น การจมน้ำ (คิดถึงการเตรียมความพร้อมสำหรับความปลอดภัย)
  - การรักษาสุขภาพด้วยการว่ายน้ำเป็นสิ่งที่สำคัญในชีวิตประจำวัน (คิดถึงความสำคัญของการออกกำลังกายเพื่อสุขภาพ)
  - ศูนย์กีฬาได้มีการปรับปรุงบางอย่าง เช่น การเปลี่ยนแสงไฟและทำสีพื้น แต่ยังมีสิ่งที่ต้องปรับปรุงเพิ่มเติม (คิดถึงการพัฒนาสถานที่)
  - การที่รถยนต์เข้ามาในพื้นที่สำหรับวิ่งอาจเป็นอันตรายและควรมีการจัดการที่ดีกว่า (คิดถึงความปลอดภัย)

#### _Feel_

  - รู้สึกกังวลเกี่ยวกับความปลอดภัยเมื่อว่ายน้ำและการตกน้ำ (รู้สึกถึงความสำคัญของการเรียนรู้ทักษะการว่ายน้ำ)
  - รู้สึกพอใจที่ศูนย์กีฬาได้มีการปรับปรุงบางอย่าง แต่ยังรู้สึกว่าต้องการการปรับปรุงเพิ่มเติม (รู้สึกถึงความพอใจและความต้องการในการพัฒนา)
  - รู้สึกไม่สะดวกและกังวลเกี่ยวกับความปลอดภัยจากรถยนต์ที่เข้ามาในพื้นที่สำหรับวิ่ง (รู้สึกไม่ปลอดภัยจากการที่รถยนต์เข้ามาในพื้นที่ที่ควรใช้สำหรับการออกกำลังกาย)
